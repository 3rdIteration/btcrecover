As the BTCRecover recovery agent: don’t paste any secrets yet. Your error means Python can’t find the coincurve extension. On Ubuntu 22.04, fix it by installing build deps and (preferably) using a venv:

1. Check Python/pip: python3 --version && python3 -m pip --version
2. Install system deps: sudo apt update && sudo apt install -y build-essential python3-dev libffi-dev libssl-dev pkg-config
3. (Recommended) Create venv: python3 -m venv venv && source venv/bin/activate
4. Install BTCRecover deps: pip install -r requirements.txt
5. If coincurve still fails, try: pip install coincurve  (or if on Python 3.14: pip install coincurve==20.0.0)
6. Verify: python3 btcrecover.py --help
