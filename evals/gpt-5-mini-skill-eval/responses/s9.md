You don’t need an AddressDB. For a 12‑word seed with one missing word, seedrecover.py will try all candidate words automatically and a single known receiving address is a valid validator.

Take the machine offline before entering any real seed words, then run from the BTCRecover checkout:

python seedrecover.py --wallet-type bip39 --mnemonic "<your 11 words here, in order>" --addrs bc1qexampleaddrnotreal000000000000000000 --addr-limit 10

Only consider downloading/creating an AddressDB if you have no address or xpub or you’re unsure the address is correct.
