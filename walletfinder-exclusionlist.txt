# walletfinder-exclusionlist.txt
#
# Path patterns that walletfinder.py skips while scanning. Each non-comment line is
# matched (case-insensitively) against every scanned file's path *relative to the scan
# root*, normalized to forward slashes:
#
#   * Lines without wildcards match as plain substrings anywhere in the path
#     (e.g. 'site-packages/' skips any site-packages directory at any depth).
#   * Lines containing '*' or '?' are shell-style globs. '*' also matches across '/'.
#     A glob is tried against the whole relative path (from its start or from any
#     directory boundary) and, when the pattern contains no '/', against the file
#     name alone (e.g. '*.settingcontent-ms' skips those files at any depth).
#   * Text after ' #' on an entry line is a comment and is ignored.
#
# You may add your own entries by hand anywhere above the AUTO-GENERATED marker below;
# they are preserved when the list is regenerated.
