# Repository Instructions

## Code Quality
- Ensure that all code changes conform to PEP 8 guidelines, including converting any tab-based indentation to spaces.

## Required Tests
- Always run `python run-all-tests.py` before completing work on this repository.

## AI-Assisted Recovery Skill
- If you are an AI agent helping a user **recover a wallet** with BTCRecover (rather than making code changes), follow the workflow in [`skills.md`](skills.md) at the repository root.
- For installation guidance on where to drop `skills.md` for Claude Code, GitHub Copilot, ChatGPT or Cline, see [`docs/AI_Assisted_Recovery.md`](docs/AI_Assisted_Recovery.md).
